# 멍냥 (MeongNyang) — 앱 디자인 설계서

## 브랜드 아이덴티티

**컨셉:** 반려동물 보호자를 위한 신뢰할 수 있는 장소 정보 플랫폼. 강남언니/강남미인처럼 실제 이용자 리뷰 기반의 정보 신뢰성을 핵심 가치로 삼는다.

**컬러 팔레트:**
- Primary: `#C8956C` (따뜻한 브라운 — 강아지/고양이 털 색감)
- Background: `#FAF7F2` (크림 베이지 — 따뜻하고 깨끗한 느낌)
- Surface: `#FFFFFF` (카드 배경)
- Secondary: `#E8D5C4` (연한 베이지 — 서브 배경)
- Accent: `#A0522D` (딥 브라운 — 강조)
- Foreground: `#2C1810` (다크 브라운 — 본문 텍스트)
- Muted: `#8B6F5E` (미디엄 브라운 — 보조 텍스트)
- Success: `#6BAF6B` (그린)
- Error: `#D9534F` (레드)
- Border: `#E8D5C4` (연베이지 보더)

**타이포그래피:** 시스템 폰트 기반, 헤드라인은 굵게, 본문은 가독성 중심

## 주요 화면별 콘텐츠 및 기능

### S01 홈 (메인)
- **헤더:** 앱 로고 "멍냥" + 검색 아이콘 + 알림 아이콘
- **카테고리 바로가기:** 카페, 식당, 동물병원, 미용실, 유치원, 호텔/위탁, 놀이터/공원, 펫샵 (2줄 4개씩 아이콘 그리드)
- **인기 장소 섹션:** 수평 스크롤 카드 리스트 (별점 높은 순)
- **내 주변 장소 섹션:** 위치 기반 가까운 장소 카드
- **리뷰 많은 장소 섹션:** 리뷰 수 기준 인기 장소

### S02 지도 탐색
- **전체 화면 지도** (react-native-maps)
- **상단 카테고리 필터 탭** (수평 스크롤)
- **하단 장소 카드 패널** (슬라이드업 시트)
- **현재 위치 버튼**
- **정렬 옵션** (거리순/평점순/리뷰순)

### S03 커뮤니티
- **게시판 카테고리 탭:** 질문, 산책친구, 후기, 자유
- **게시글 리스트** (FlatList)
- **글쓰기 FAB 버튼**

### S04 마이페이지
- **프로필 헤더** (아바타, 닉네임, 이메일)
- **내 반려동물 프로필 카드** (등록된 반려동물 목록)
- **메뉴 리스트:** 찜한 장소, 내 리뷰, 설정
- **로그인/로그아웃 버튼**

## 핵심 사용자 플로우

### 플로우 1: 장소 탐색 → 상세 → 리뷰 확인
홈 → 카테고리 탭 → 장소 리스트 → 장소 상세 → 리뷰 탭 확인

### 플로우 2: 지도 기반 탐색
지도 탭 → 현재 위치 확인 → 카테고리 필터 선택 → 마커 클릭 → 장소 상세

### 플로우 3: 리뷰 작성
장소 상세 → 리뷰 작성 버튼 → 로그인 확인 → 별점/태그/사진 입력 → 제출

### 플로우 4: 커뮤니티 참여
커뮤니티 탭 → 카테고리 선택 → 게시글 읽기 or 글쓰기 → 댓글 작성

### 플로우 5: 반려동물 프로필 등록
마이페이지 → 반려동물 추가 → 이름/종류/견종 입력 → 저장
